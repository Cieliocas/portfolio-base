Portfolio pessoal com foco em HPC, DevOps, Cloud e AI Infrastructure.

## Visao Geral

Este projeto foi reconstruido em Next.js com uma identidade visual neon/cyber, layout responsivo e modo light/dark.

Principais pontos:
- Hero com efeito de digitacao no nome.
- Menu rapido fixo inferior para navegacao entre secoes.
- Secao "Sobre" com estetica inspirada em Notas do macOS.
- Secao de projetos com links para repositorios.
- Contato com atalho direto para WhatsApp.

## Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Lucide Icons

## Estrutura

- `app/`: layout, pagina principal e estilos globais.
- `components/`: secoes do portfolio e componentes reutilizaveis.
- `public/`: assets estaticos (foto, favicon, curriculo).

## Como Rodar Localmente

```bash
npm install
npm run dev
```

Acesse: `http://127.0.0.1:3000`

## Scripts

- `npm run dev`: ambiente de desenvolvimento.
- `npm run build`: build de producao.
- `npm run lint`: analise de lint.
